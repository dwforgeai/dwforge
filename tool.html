Tips for better results

    
    
      
  • Use real field names from your actual system. Generic names give generic output.
    • 
      
  • Say "single record" or "batch of records" so DWForge structures the code correctly.
    • 
      
  • Mention null rules. For example: skip the record if ID is null, or default amount to 0.
    • 
      
  • Specify date formats both ways. What format is it coming in, and what does the target expect.
    • 
      
  • For Salesforce sources, include relationship fields like Account__r.ERP_ID__c in your description.
